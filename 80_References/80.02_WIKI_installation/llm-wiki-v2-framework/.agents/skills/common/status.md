# Common Completion Status

## Complete

All work required by the selected mode and resolved scope was completed.

## Partial

Meaningful work completed, but some files, candidates, or checks were deferred due to scope, limits, ambiguity, tooling, or unreadable content.

## Skipped

The target required no processing, such as an unchanged already-ingested source.

## Failed

No meaningful work could be completed because the target was missing, unreadable, or unwritable.

Do not use `Failed` when partial useful work exists.
