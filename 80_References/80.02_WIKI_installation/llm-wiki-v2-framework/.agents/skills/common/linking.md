# Common Linking Policy

Use Obsidian-style links:

```markdown
[[Page Name]]
[[Page Name|Alias]]
[[Page Name#Heading]]
```

## Link Rules

- Add links only when they carry semantic value.
- Prefer exact canonical page titles.
- Preserve existing aliases.
- Avoid linking every keyword.
- Avoid circular link noise.
- Source pages should link to central related pages.
- Updated related pages should link back to relevant source pages.
- Do not create a page only to satisfy a weak link candidate.
- Report ambiguous link targets instead of choosing silently.
