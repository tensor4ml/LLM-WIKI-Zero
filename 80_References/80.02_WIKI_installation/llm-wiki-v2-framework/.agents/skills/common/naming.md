# Common Naming Policy

## Files

Use lowercase kebab-case for new Wiki filenames.

Examples:

```text
retrieval-augmented-generation.md
2026-07-10-example-source.md
wiki-health-check-2026-07-10.md
```

## Titles

Use clear human-readable page titles in frontmatter and H1.

## Source Files

Prefer:

```text
YYYY-MM-DD-short-source-title.md
```

Use `undated-` when no source date is available.

## Duplicate Prevention

Before creating a page, check:

1. filename
2. title
3. aliases
4. index entries
5. narrowed search results
