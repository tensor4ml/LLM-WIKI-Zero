# Common Language Policy

Use the primary language of the user's latest explicit request.

- Korean request: answer and create artifacts in Korean.
- English request: answer and create artifacts in English.
- Mixed request: use the language carrying the main instruction.

Keep these in their original form:

- file paths
- filenames
- frontmatter keys
- code identifiers
- commands
- product and model names
- quoted source titles
- proper nouns where translation would reduce precision

Do not mix heading languages without a clear reason.
