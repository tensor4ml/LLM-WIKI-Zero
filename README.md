# LLM Wiki

이 저장소는 Markdown 기반의 LLM Wiki입니다.

Codex, Claude, Antigravity와 함께 실행됩니다.

핵심 에이전트 지침은 `AGENTS.md`, `GEMINI.md`, `CLAUDE.md` 에 있습니다. 상세 운영 규칙은 `30_Rules/` 아래에 있으며, 생성된 지식은 `20_Wiki/` 아래에 저장됩니다. 변경하지 않는 원본 자료는 `10_Raw/` 아래에 배치해야 합니다.

# 설치

1. AI 로 `LLM Wiki Prompt.md`를 실행합니다. 이 작업은 LLM Wiki에 필요한 폴더, 파일 및 AI를 위한 스킬들을 생성합니다.
```
LLM Wiki Prompt.md 를 읽고 지시들을 수행합니다. 
```

2. `80_References/80.02_WIKI_installation/graphify-installation.md`의 설치 지침을 읽고 따라 진행합니다.
3. 사용법은 `LLM-Wiki-Skills-Usage-Guide.md`를 참고합니다.
